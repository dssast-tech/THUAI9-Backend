这个目录已按“当前可用架构 + 现在要做什么”重新整理，命名采用统一编号：

- `00_文档导航.md`：你现在看到的导航。
- `01_当前架构总览.md`：从入口到 UI 的完整流程图和关键路径。

1. 先看 `01_当前架构总览.md`，对全局有感觉。
2. 再看 `02_核心模块说明.md`，按文件定位代码。
3. 最后看 `03_当前待做清单.md`，直接挑任务开做。
